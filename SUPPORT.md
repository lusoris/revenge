## Response Times

We try to respond to issues and discussions as quickly as possible:

| Type | Target Response Time |
|------|---------------------|
| Security Issues | 48 hours |
| Bug Reports | 1 week |
| Feature Requests | 2 weeks |
| General Questions | 1 week |

Note: These are targets, not guarantees. This is an open-source project maintained by volunteers.

## Community Guidelines

- Be respectful and constructive
- Provide as much context as possible
- Search before asking
- Help others when you can
