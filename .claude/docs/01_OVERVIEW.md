## Get Started in 30 Seconds

**Prerequisites**: Claude subscription (Pro, Max, Teams, or Enterprise) or Claude Console account

**Install:**
```bash
# macOS, Linux, WSL
curl -fsSL https://claude.ai/install.sh | bash

# Windows PowerShell
irm https://claude.ai/install.ps1 | iex
```

**Start:**
```bash
cd your-project
claude
```

## What Claude Code Does

- **Build features from descriptions**: Tell Claude what you want in plain English
- **Debug and fix issues**: Describe bugs or paste error messages
- **Navigate any codebase**: Ask about project structure, find info from web, use MCP for external sources
- **Automate tedious tasks**: Fix lint issues, resolve merge conflicts, write release notes

---

## Why Developers Love Claude Code

- **Works in your terminal**: Not another chat window or IDE
- **Takes action**: Directly edit files, run commands, create commits
- **Unix philosophy**: Composable and scriptable
- **Enterprise-ready**: Use Claude API, host on AWS/GCP, built-in security

## Use Claude Code Everywhere

| Platform | Description |
|----------|-------------|
| Terminal (CLI) | Core experience - run `claude` |
| Claude Code on web | Use at claude.ai/code, no local setup |
| Desktop app | Standalone with diff review, parallel sessions |
| VS Code | Native extension with inline diffs |
| JetBrains IDEs | Plugin for IntelliJ, PyCharm, WebStorm |
| GitHub Actions | Automate code review with @claude mentions |

| Slack | Route coding tasks via mentions |
| Chrome | Live debugging and design verification |
