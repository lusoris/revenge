# Accessibility




> Accessible to everyone, by design

Revenge targets WCAG 2.1 AAA, the highest accessibility standard. Color contrast meets 7:1 for normal text. Full keyboard navigation with visible focus states. Screen reader support with ARIA labels and live regions. Subtitle customization includes font choice, colors, sizing, and position. Respects prefers-reduced-motion for users sensitive to animation.
