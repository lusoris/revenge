# StashDB




> StashDB metadata - Supplementary to Whisparr

SUPPLEMENTARY metadata source for QAR (adult) content. Used as: (1) **Fallback** when Whisparr not setup, unreachable, or lacks metadata, (2) **Enrichment** for additional data. Community-maintained database (performers, studios, scenes). Fingerprinting for file matching. GraphQL API. **Optional proxy/VPN routing** for privacy (via HTTP_CLIENT service).
