3. Select Styling

Section titled 3. Select Styling

A layout refers to the arrangement and presentation of various player components. The CSS and
Tailwind CSS options below are if you want to style components from scratch and build your own
layout. The Default Theme option is if you want to build your own layout on top of our component
styles. Finally, the Default and Plyr layouts are our production-ready templates to get you up and
running quickly.
